# TakeOne — Visão do Produto

## One-liner

Cronograma inteligente para streamers de reação + hub de descoberta de filmes/séries.

## O Problema

O projeto original (filmes-novos) era apenas um cliente TMDB — bonito visualmente, mas sem utilidade real. Exibe dados que qualquer outro site já exibe.

Duas dores identificadas:

1. **Streamers de reação** — não têm uma ferramenta simples pra criar e divulgar sua grade semanal de programação. Muitos usam Trello, Google Sheets, ou criam sites próprios do zero (ex: mahmoojen.com.br/cronograma).
2. **Fãs de streamers** — não existe um lugar centralizado pra acompanhar a programação dos seus streamers favoritos. Precisam seguir cada um em cada plataforma separadamente. Quando querem salvar um filme pra ver depois, usam Instagram, WhatsApp ou a própria cabeça.

## Pilares do Produto

| Pilar | Pra quem | O que resolve |
|-------|----------|---------------|
| Cronograma de Streamers | Streamers + Fãs | "Onde divulgo/acompanho minha grade semanal?" |
| Watchlist Pessoal | Público geral | "Quero salvar esse filme que o streamer vai reagir" |
| Hub de Reações (futuro) | Público + Streamers | "Onde acho reações desse filme?" |

## Diferencial Competitivo

| Concorrente | O que faz | Não faz |
|-------------|-----------|---------|
| Letterboxd | Reviews sociais | Nada de streamers/cronograma |
| TMDB | Banco de dados | Nada de streamers/cronograma |
| JustWatch | Onde assistir | Nada de streamers/cronograma |
| Sites próprios de streamers | Cronograma próprio | Cada um tem o seu, isolado |
| **TakeOne** | Cronograma + TMDB + Watchlist + Reações | — |

## Monetização Potencial (futuro)

- Streamer verificado (badge + personalização extra do embed)
- Destaque no "Ao Vivo Agora" (posicionamento pago)
- Embed sem branding (plano pro para streamers)
- Links de afiliado de streaming
